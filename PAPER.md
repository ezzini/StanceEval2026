# Paper Submission Guidelines

## Overview
Shared task papers must follow the format for **short papers at ArabicNLP 2026**.

## Paper Format
- Papers must follow official **EMNLP (ACL) LaTeX or Word templates**
- Available at: [https://github.com/acl-org/acl-style-files](https://github.com/acl-org/acl-style-files)
- **Length**: Up to 4 pages (excluding unlimited references and appendices)

## Paper Title Format
Use this format:
> `<Team Name> at <Task Name>: <Your Contribution>`

**Example**: "Scholarly at AraGenre 2026: Hierarchical Genre Classification via Few-Shot Prompting"

## Submission System
- All papers must be submitted via **OpenReview**
- An OpenReview account is required for all participants and organizers
- Papers are **not anonymous** (shared task papers are non-blind)

## Requirements
- Papers must cite the shared task overview paper
- The main content must not exceed **4 pages**
- No page limit on appendices and references
- At least one author must register for ArabicNLP 2026 / EMNLP 2026

## Important Dates
- **August 22, 2026**: Camera-ready system description papers due
- **September 1, 2026**: Shared task overview papers due
- **September 10, 2026**: Conference camera-ready deadline

## Additional Notes
- Authors are free to post papers on arXiv at any time
- No revisions or name changes allowed after the review period
- If research raises ethical concerns, discuss them in the paper
- Release of code and data is strongly encouraged for reproducibility

## Questions?
Contact the shared task organizers or the ArabicNLP 2026 workshop organizers.
